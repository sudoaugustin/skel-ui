## Skeletons at foundation

Implement skeletons at the foundation of your application. By default, it will render a real UI, but when wrapped inside a `Root`, it will behave like a skeleton.

<Portal className="h-72" disableRefresh>
    <Buttons>
        ```tsx
        <Button label="Press Me"/>
        ```
        ```tsx
        <Skel.Root>
          <Button label="Press Me"/>
        </Skel.Root>
        ```
    </Buttons>
</Portal>

```tsx title="Button.tsx"
export default function Button({ label, ...rest }) {
  return <Skel.button {...rest}>{label}</Skel.button>;
}
```
